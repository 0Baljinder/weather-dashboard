# Weather Dashboard

A responsive weather dashboard built using HTML, CSS, and JavaScript. The application fetches real-time weather data using the Open-Meteo API and displays current weather information for any searched city. It also displays a random motivational quote to enhance the user experience.

---

## Features

- Search weather by city name
- Display current temperature
- Display daily high temperature
- Display daily low temperature
- Display wind speed
- Display current weather condition
- Display the current date
- Remember the last searched city using Local Storage
- Show a loading spinner while fetching weather data
- Display a random motivational quote
- Responsive layout for desktop and mobile devices

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- Local Storage

---

## APIs Used

### Open-Meteo Geocoding API

Converts a city name into latitude and longitude.

https://geocoding-api.open-meteo.com

### Open-Meteo Forecast API

Provides current weather and daily forecast data.

https://open-meteo.com/

### DummyJSON Quotes API

Provides random motivational quotes.

https://dummyjson.com/quotes
